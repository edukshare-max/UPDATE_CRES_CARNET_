MANUAL DE USUARIO - CRES CARNETS UAGro
====================================================================

¡BIENVENIDO!

Gracias por instalar CRES Carnets, el sistema de gestión de carnets
de salud digital de la Universidad Autónoma de Guerrero.

PRIMER USO:
----------
1. Al abrir la aplicación por primera vez, deberá iniciar sesión
2. Ingrese su nombre de usuario y contraseña proporcionados por
   la Dirección de Innovación en Salud
3. Seleccione su campus/institución
4. Haga clic en "Iniciar Sesión"

FUNCIONALIDADES PRINCIPALES:
---------------------------

📋 CARNETS DIGITALES:
   - Crear y gestionar carnets estudiantiles
   - Búsqueda rápida por matrícula
   - Registro fotográfico
   - Historial completo

🏥 ATENCIÓN MÉDICA:
   - Registro de consultas
   - Notas médicas
   - Seguimiento de casos
   - Historial clínico

📊 PROMOCIÓN DE SALUD:
   - Campañas de prevención
   - Estadísticas
   - Reportes

MODO OFFLINE:
------------
La aplicación funciona incluso SIN CONEXIÓN A INTERNET:
- Los datos se guardan localmente
- Al recuperar conexión, se sincronizan automáticamente
- Indicador visual del estado de conexión

ATAJOS DE TECLADO:
-----------------
- Ctrl + N: Nuevo carnet
- Ctrl + B: Buscar carnet
- Ctrl + S: Guardar cambios
- F5: Actualizar datos
- Esc: Cerrar ventana actual

ACTUALIZACIÓN AUTOMÁTICA:
------------------------
El sistema verifica actualizaciones al iniciar:
- Si hay una nueva versión, se notificará automáticamente
- La actualización se descarga e instala automáticamente
- No perderá sus datos durante la actualización

SOLUCIÓN DE PROBLEMAS:
---------------------

Problema: No puedo iniciar sesión
Solución: 
  1. Verifique su usuario y contraseña
  2. Asegúrese de tener conexión a internet
  3. Verifique que seleccionó el campus correcto
  4. Contacte al administrador si persiste

Problema: Los datos no se sincronizan
Solución:
  1. Verifique su conexión a internet
  2. Presione F5 para forzar sincronización
  3. Revise el indicador de estado en la barra superior

Problema: La aplicación está lenta
Solución:
  1. Cierre y vuelva a abrir la aplicación
  2. Verifique que no haya muchas aplicaciones abiertas
  3. Libere espacio en disco si está lleno

DESINSTALAR:
-----------
Para desinstalar la aplicación:
1. Vaya a Configuración > Aplicaciones
2. Busque "CRES Carnets UAGro"
3. Haga clic en "Desinstalar"
4. Sus datos en la nube permanecerán seguros
